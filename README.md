## Краткое описание архитектуры
Клиент на go - *handler* обрабатывает новые сообщения (get updates long polling),
передаёт голосовые сообщения *api* микросервиса языковой модели на python использующей открытую библиотеку whisper. Модель обрабатывает аудио и возвращает расшифровку, *handler* отправляет сообщение, отвечающее на голосовое, с его расшифровкой<br>
## Локальный запуск 
Запустите сборку контейнеров командой
` make `

```bash
 make
 make handler   ##  Собрать только контейнер для обработчика api max
 make api       ##  Собрать только контейнер для api языковой модели
```
Запустите контейнер с *api*, затем контейнер с *handler*, "token" - ваш токен
```bash
docker run -p 8080:80 -d api 
docker run -e TOKEN="token" --net=host -d handler
```
